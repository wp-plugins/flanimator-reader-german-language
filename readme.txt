German Post Reader - Flanimator

== Description ==

Flanimator Reader provides speech-synthesis for your blog posts. In this early stage of development, the reader is limited to German language although English language is in development, too.

== Installation ==

1. Upload the .php to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Nothing else ;-)
